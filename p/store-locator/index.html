Store Locator Webapp

Overview

This webapp showcases point of interest locales to market, in this instance, a grocery chain's closest stores to a user. I was responsible for building a mobile-friendly front-end template that can be easily tailored to match a client's branding, while driving users to sign up for the loyalty club with a coupon incentive call to action.

Role

Strategy, user research, product design, development

Tools

HTML, CSS, JS, PHP, Google Maps API
